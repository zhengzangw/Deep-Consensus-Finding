## Description

Deep neural network to find the consensus of clustering in DNA storage system

## How to run

Install dependencies

```bash
pip install -r requirements.txt
```

Build dataset
```bash
python -m src.generate_DNA
```

Run scripts

```bash
python -m src.main
python -m src.inference
```
